Welcome to the pubicly shared catalogue of concept and datamodels

Here you'll find an overview of the concept and datamodels created in the prublic regime and which are registered with a foucs on knowledge sharing and reuse. Furthermore this catalogue contains information about a number of internationally regonized models, which can have a broad use in the Danish administrative and public context. Use of the modelrules and reuse of models and data enables us to create a publicly shared data architecture and a common data... where we can use eachothers models and the data they document.
